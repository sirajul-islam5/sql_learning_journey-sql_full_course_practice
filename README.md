# SQL Mastery: A Comprehensive Learning Journey
This repository contains a structured, 5-part series of SQL scripts developed during an intensive training marathon. It covers the full spectrum of SQL skills, from foundational data definition to complex analytical querying.

## 📁 Repository Structure
The project is divided into five logical modules for better navigation and learning:

1. **[01_DDL_and_DML_Basics.sql](./01_DDL_and_DML_Basics.sql)**
   * Focuses on Database/Table creation and data insertion.
   * Covers Constraints: Primary Key, Foreign Key (with Cascading), Check, and Default values.

2. **[02_Query_Fundamentals.sql](./02_Query_Fundamentals.sql)**
   * Explores data retrieval techniques using `SELECT` and `WHERE`.
   * Covers Logical Operators (`AND`, `OR`), Range filtering (`BETWEEN`, `IN`), and Sorting (`ORDER BY`).

3. **[03_Aggregations_and_Grouping.sql](./03_Aggregations_and_Grouping.sql)**
   * Demonstrates data summarization using Aggregate functions (`SUM`, `AVG`, `COUNT`).
   * Implements `GROUP BY` and `HAVING` clauses for business insights.

4. **[04_Joins_and_Subqueries.sql](./04_Joins_and_Subqueries.sql)**
   * Deep dive into relational logic: `INNER`, `LEFT`, `RIGHT`, and `SELF JOIN`.
   * Covers nested logic with `SUBQUERIES` and creating `VIEWS`.

5. **[05_Advanced_Operations_and_Exercises.sql](./05_Advanced_Operations_and_Exercises.sql)**
   * Covers Data Maintenance (`UPDATE`, `DELETE`) and Schema Evolution (`ALTER TABLE`).
   * Includes final capstone exercises and practical case studies.

## 🛠️ Tech Stack
* **Language:** SQL (Structured Query Language)
* **Environment:** MySQL Workbench
* **Key Skills:** Database Design, Data Integrity, Relational Logic, and Analytical Querying.
